## How to build and run the application?

1. Clone the repository
```
git clone <your-repo-url>
cd braintree-spa-checkout
```
2. Install dependencies
```
pip install -r requirements.txt
```
3. Start the server
```
python app.py
```
4. Go to the following url in your browser:
```
http://127.0.0.1:8080/
```
